# E-COMMERCE-WEBSITE-
A simple web form built using HTML, CSS, and JavaScript with basic validation for name and email. The form ensures both fields are filled and the email contains "@". A Javascript is also included that takes user input (name and email) and prints the received data.

## Features
- Sticky navbar with category dropdowns
- Form validation (login/signup/contact)
- Dynamic feedback slider with autoplay
- Fully responsive across all devices

## Tech Stack
HTML5 | CSS3 | Vanilla JavaScript
